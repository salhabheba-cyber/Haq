## 📋 What is Haq?

**Haq** is a professional anti-scam platform that helps users detect fraudulent messages on:
- 📱 **WhatsApp**
- ✉️ **SMS**
- 📞 **Phone Calls**

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI Analysis** | Uses OpenAI GPT to analyze message content |
| 🌍 **Bilingual** | Reports in Arabic and English |
| 🔗 **URL Scanner** | Detects malicious links |
| 📞 **Phone DB** | Checks numbers against spam database |
| 📊 **Admin Dashboard** | Track user searches and statistics |
| 📥 **CSV Export** | Export search history for analysis |

## 🛠️ Installation (For Developers)

```bash
# Clone the repository
git clone https://github.com/salhabheba-cyber/Haq.git
cd Haq

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run web/app.py
