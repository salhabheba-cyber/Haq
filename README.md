# 🛡️ Haq - اعرف الحَقّ قبل ما تندم

**🇱🇧 Built in Beirut, Lebanon**

## What is Haq?
Haq helps you detect scam messages on WhatsApp, SMS, and phone calls.

## Features
- 📱 WhatsApp message scanner
- ✉️ SMS spam detector
- 📞 Phone number checker
- 🌍 Bilingual reports (Arabic/English)
- 📊 Admin dashboard
